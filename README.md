# Base de projets JSP/Java EE

Ce dossier est la base commune de tes projets du CDC.

## Projets disponibles
- `projet-1-tournois` : gestion de tournois sportifs.
- `projet-2-textile` : gestion des commandes textile avec workflow.
- `projet-3-taxis` : gestion flotte de taxis, courses et revenus.
- `projet-4-parking` : gestion de parking (places, reservations, entrees/sorties).
- `projet-5-livraison` : service de livraison (commandes, affectation, suivi).
- `Exercice - Projet Simple - CDC.pdf` : cahier des charges.

Chaque sous-dossier garde sa propre structure Maven, son propre `README.md`, et son propre `sql/schema.sql`.
